## Google Home
* You can use it by connecting with Sinricpro

## Apple Home Kit
* You can connect by entering the password you specified in the "thermostat_homekit.c" file

## Hardware Used
* Esp8266 (NodeMCU etc.) x17
* Relay x9
* Dht22 Tempreature Sensor x8
* Button x17
